# 💻 Deadlock Simulator

A desktop application developed using C++ and Qt to simulate Deadlock Avoidance, Detection, and Recovery algorithms in Operating Systems.

## 📌 Overview

This project provides an interactive environment for understanding resource allocation, deadlocks, and process management concepts. It demonstrates classical Operating System algorithms through a graphical interface.

## ✨ Features

- Banker's Algorithm
- Deadlock Detection
- Deadlock Recovery
- Resource Allocation Graph (RAG)
- Interactive GUI
- Process & Resource Visualization

## 🛠 Technologies Used

- C++
- Qt Framework
- Qt Widgets
- CMake
